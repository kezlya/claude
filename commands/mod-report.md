# Mod Report and Ed Sync Prep

You are preparing Vitaly's weekly sync with Ed, the lead mod.

The sync is 20 minutes. It should cover what's working, what's not, and what needs to change - not just a status update.

## Step 1: Get Mod Activity Update

Ask Vitaly:
> "Quick mod update before the Ed sync. What did you observe from each mod this week - what they posted, conversations they started, any scouting activity they mentioned? Even rough impressions count."

Wait for response. If Vitaly is unsure about specific mods, ask:
> "Even a general read - were they active? Did anything they did create visible community impact?"

## Step 2: Evaluate Performance (use content-mod-tracker agent)

Run the **content-mod-tracker** agent with what Vitaly described. Let it produce the full performance report and Ed sync agenda.

## Step 3: Reality Check the Performance

Run a quick content-validator check: are the performance expectations realistic given that the community is pre-launch or early stage? Adjust if needed.

## Step 4: Present

```
MOD REPORT - [date]
===================
[Full content-mod-tracker output]

ED SYNC AGENDA:
[Prioritized list of what to cover, with specific questions]
```

Ask:
> "Does this match your read? Anything I'm being too hard or too easy on?"

## Step 5: Sync Logistics

Remind Vitaly: the sync is 20 minutes. Set expectations:
- Start with one thing that went well (1 min)
- Cover the top concern directly (10 min)
- Agree on one specific outcome for next week (5 min)
- Remaining topics (4 min or async)

If there's a performance issue that needs addressing, offer to help draft the specific language Vitaly should use with Ed.
