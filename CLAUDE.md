## Projects

All local repositories live under `~/code/` (the `code` folder in the user's home directory on any machine).

A full project catalog is maintained at:
- **Source of truth:** `~/code/kezlya/ai/projects.yaml`
- Contains all project groups, repo paths, descriptions, and git origin URLs
- When setting up a new machine, clone repos from the origins listed in that file
